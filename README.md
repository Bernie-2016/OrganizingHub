# OrganizingHub

Static landing site for Bernie 2016 organizing.

## Development

### Prerequisites

* node and npm

### Setup

1. Clone the repository (`git clone git@github.com:Bernie-2016/OrganizingHub.git`)
2. Install packages: `npm install`
3. Run `gulp serve` and open `dist/index.html`.
